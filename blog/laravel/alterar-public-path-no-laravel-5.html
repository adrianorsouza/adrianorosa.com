Alterar public path no Laravel 5

09 May 2015 — Laravel — 0 Comentário

No Laravel 5 o arquivo bootstrap/paths.php foi removido, antes era possível alterar os diretórios padrão usado pelo framework, ou seja, quando queríamos definir outro diretório público do webserver para outro nome diferente de public bastava definir esse valor no array para por exemplo public_html.

Mas desde a versão 5 do Laravel, e até o momento que escrevo este post, não há uma maneira simples de alterar esses paths, então o que você deve fazer é extender a classe Illuminate\Foundation\Application e redefinir o método publicPath() ou qualquer outro path que deseja modificar ex. o diretório storage deverá redefinir o método storagePath(), após criar a classe também é necessário alterar o arquivo bootstrap/app.php onde a classe é instanciado e utilizado em toda aplicação.

Sua classe ficaria assim:

namespace MyApp
class Application extends \Illuminate\Foundation\Application {
    public function publicPath()
    {
        return $this->basePath.DIRECTORY_SEPARATOR.'public_html';
    }
}

No arquivo bootstrap/app.php instancie a classe Application dentro do namespace da sua aplicação:

$app = new MyApp\Application(
    realpath(__DIR__.'/../')
);

Existe também outras maneiras de alterar os paths, como por exemplo redefinir helper function public_path() no seu index.php, como no exemplo logo abaixo retirado do fórum laracasts, mas não recomendo esse outro tipo de hack:

function public_path($path = '')
{
    return realpath(__DIR__);
}
